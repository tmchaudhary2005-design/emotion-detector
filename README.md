# Emotion Detection Application

## Project Description
This is a web-based Emotion Detection application built using Python and Flask. It utilizes the Watson NLP library to analyze text input and determine the underlying emotions. 

## Features
* Detects multiple emotion scores: Anger, Disgust, Fear, Joy, and Sadness.
* Identifies the dominant emotion from the text.
* Includes a web interface built with Flask.
* Features error handling for blank inputs and invalid API responses (Status Code 400).
* Unit tested and verified with static code analysis (Pylint score 10/10).

## Technologies Used
* Python
* Flask
* IBM Watson NLP API
* HTML/CSS
